## 🏰 核心玩法：领地与传送

### 1. 领地系统（圈地保护）

#### 前期准备



* 选区工具：木锄头（默认工具，用于选择领地范围）

* 信息工具：线（查看领地归属、范围等信息）

#### 圈地步骤



1. 手持木锄头，左键点击领地区域第一个角落（选点 1）

2. 右键点击对角位置（选点 2），形成矩形领地范围

3. 输入指令：`/res create 领地名称`（创建成功）

#### 常用领地指令



* `/res list`：查看自己的所有领地（最多可创建 4 个）

* `/res tp 领地名`：传送到指定领地

* `/res remove 领地名`：删除不需要的领地

### 2. 传送功能指令



| 指令               | 功能说明             | 使用示例           |
| ---------------- | ---------------- | -------------- |
| `/tpa 名>`        | 向指定玩家发送传送请求      | `/tpa Steve`   |
| `/tphere 名>`     | 将指定玩家传送到自己身边     | `/tphere Alex` |
| `/tpaccept`      | 接受传送请求           | -              |
| `/tpdeny`        | 拒绝传送请求           | -              |
| \`/tp    传送到指定坐标 | `/tp 100 64 200` |                |

## 💰 经济系统：积分获取与消费

T Craft 以「积分」为核心货币，支持签到、交易、购物等多种玩法，经济系统持续平衡，无超模氪金奖励。

### 1. 积分获取方式



| 获取途径 | 操作方式       | 奖励说明                   |
| ---- | ---------- | ---------------------- |
| 每日签到 | 主菜单 → 每日签到 | 默认 10 积分 / 天，每日限 1 次   |
| 在线奖励 | 保持游戏在线     | 默认 5 积分 / 小时，自动发放      |
| 物品回收 | 主菜单 → 物品回收 | 将可回收物品放入回收箱兑换积分，支持一键回收 |

### 2. 积分消费渠道

#### 积分商店



* 入口：主菜单 → 积分商店

* 功能：使用积分购买管理员上架的商品，点击即可完成交易

#### 全球市场



* 入口：主菜单 → 全球市场

* 买家操作：浏览店铺列表，点击商品直接购买，物品自动入背包

* 卖家操作：我的商店 → 上架物品 → 设置价格 → 确认上架（最多上架 10 个商品）

### 3. 积分转账



* 指令：`/tcraft jf   示例：`/tcraft jf Steve 100\`（给 Steve 转账 100 积分）

* 注意：仅支持在线玩家转账，操作不可撤销，需确认玩家名称正确
